# Critique rubric (this site)

The critic reads this on top of the generic doctrine in the refsite-runner
skill's critique-spec. This file says what THIS site's chapters must get right;
the spec says how to judge and when to block. Keep it short. Where this file
and the spec disagree on emphasis, this file wins.

## What to check hardest here

1. **Claims against sources.** This is an idea-driven reference site: the
   highest-value check is fetching the listed sources and confirming the
   chapter says what they say. Prefer verifying two claims deeply over
   skimming ten.
2. **The widget teaches the thesis.** Drive the widget's logic in the source:
   its state transitions must demonstrate the chapter's central claim, not a
   neighboring one.
3. **The figure's structure is true.** Orderings, magnitudes, and labels must
   survive comparison with the prose and the sources.

## Site-specific tolerances

None beyond the spec's defaults. (Add entries here when this site develops
recurring judgment calls, e.g. "population figures may be rounded to the
nearest million".)
